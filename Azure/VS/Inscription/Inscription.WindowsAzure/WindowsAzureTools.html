Kit de développement logiciel Windows Azure SDK pour .NET

Pour activer la prise en charge afin de générer des applications et des services Web évolutifs sur Windows Azure, installez le Kit de développement logiciel Windows Azure SDK pour .NET.

Le Kit de développement logiciel Windows Azure SDK pour .NET étend Visual Studio pour permettre la création, la génération, le débogage, l'exécution et la création de packages d'applications et de services Web évolutifs sur Windows Azure.

Remarque : le projet qui contient cette page fournit uniquement des instructions pour installer l'ensemble d'outils requis.
